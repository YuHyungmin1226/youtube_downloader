YouTube Downloader v2.0.0 - 설치 가이드
==========================================

📦 배포 내용
-----------
- YouTube_Downloader.exe: 메인 실행 파일 (약 50MB)
- README.md: 사용 설명서

🚀 설치 및 실행
-------------
1. 압축 파일을 원하는 폴더에 압축 해제
2. YouTube_Downloader.exe를 더블클릭하여 실행
3. 관리자 권한으로 실행 권장 (FFmpeg 설치 시)

⚙️ 시스템 요구사항
----------------
- Windows 10/11 (64비트)
- 인터넷 연결
- 최소 100MB 여유 공간

🔧 주요 기능
-----------
✅ 간편한 GUI 인터페이스
✅ 자동 FFmpeg 설치
✅ 실시간 진행률 표시
✅ 고품질 MP4 다운로드
✅ 오디오만 다운로드 옵션
✅ 자막 다운로드 (한국어)
✅ 설정 시스템
✅ 재시도 메커니즘

📖 사용 방법
-----------
1. YouTube 링크 입력
2. FFmpeg 설치 (필요시)
3. 설정 조정 (선택사항)
4. 다운로드 시작
5. 완료 후 폴더 확인

🛠️ 문제 해결
------------
- FFmpeg 설치 후 PC 재시작 권장
- 관리자 권한으로 실행해보세요
- 인터넷 연결 확인
- Windows Defender 예외 설정

📞 지원
-------
버그 리포트나 기능 제안은 이슈를 통해 제출해주세요.
